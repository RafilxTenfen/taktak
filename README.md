## How to run
- Requirements
  - JAVA 8
  - JDK 11
  - java bin in your path enviroment

- Run the server first
```shell
$~ make server-up
```

- Enter the TakTak directory like `cd ../github.com/RafilxTenfen/taktak/TakTak`

- Run the app
```shell
$~ java -Dfile.encoding=UTF-8 @/tmp/cp_5amd97g5269cb4zlh6pnyunsl.argfile App
```
